# OKLCH 색상 팔레트 생성기

> OKLCH 색공간을 활용해 시각적으로 균일하고 접근성이 뛰어난 색상 팔레트를 생성하는 도구

## 🎯 문제 상황

### 현재 상황
대부분의 디자이너와 개발자는 여전히 HSL이나 RGB 색공간을 사용해 색상 팔레트를 만들고 있습니다. 하지만 이런 색공간은 인간의 시각 인식과 일치하지 않아 색상 간 밝기 차이가 불균등하게 느껴집니다. 특히 접근성을 고려한 색상 조합을 만들 때 WCAG 대비비를 맞추기 어렵습니다.

### 문제점들
- **시각적 불균등성**: HSL에서 같은 Lightness 값이라도 색상에 따라 실제 밝기가 다르게 인식됨
- **접근성 예측 어려움**: WCAG AA/AAA 기준을 만족하는 색상 조합을 찾기 위해 반복적인 시행착오 필요
- **일관성 부족**: 브랜드 색상을 기반으로 한 파생 색상들이 시각적으로 일관된 밝기를 갖지 못함

### 누가 이런 문제를 겪나요?
디자인 시스템을 구축하는 프론트엔드 개발자, UI/UX 디자이너, 그리고 접근성을 중요하게 생각하는 웹 개발자들이 이런 문제에 자주 부딪힙니다.

## 💡 해결책

### 핵심 개념
OKLCH 색상 팔레트 생성기는 인간의 시각 인식에 기반한 OKLCH 색공간을 활용해 시각적으로 균일한 색상 팔레트를 생성합니다. 기준 색상을 입력하면 동일한 지각적 밝기(Lightness)를 가진 색상들로 팔레트를 만들고, 자동으로 WCAG 접근성 기준을 확인해줍니다. 생성된 팔레트는 CSS, Tailwind, 또는 디자인 토큰 형태로 내보낼 수 있습니다.

### 작동 방식
1. **기준 색상 입력**: 브랜드 색상이나 원하는 색상을 OKLCH, HSL, 또는 HEX로 입력
2. **팔레트 생성**: Lightness와 Chroma 값을 조정하여 조화로운 색상 조합 자동 생성
3. **접근성 검증**: 생성된 모든 색상 조합에 대해 WCAG AA/AAA 대비비 자동 계산
4. **내보내기**: CSS 변수, Tailwind config, JSON 형태로 팔레트 내보내기

### 주요 기능
- **실시간 시각화**: OKLCH 색공간에서 색상 관계를 3D로 시각화하여 직관적 이해 제공
- **접근성 우선 생성**: APCA와 WCAG 기준을 모두 지원하는 대비비 계산
- **다양한 내보내기**: CSS Custom Properties, Tailwind CSS, Figma Tokens 등 다양한 형식 지원
- **색맹 시뮬레이션**: Protanopia, Deuteranopia, Tritanopia 시뮬레이션으로 접근성 검증

## 🎨 흥미로운 점들

### 독특한 접근법
대부분의 색상 도구들이 HSL 기반인 반면, 이 도구는 인간의 시각 인식에 기반한 OKLCH 색공간을 활용합니다. OKLCH는 같은 Lightness 값의 색상들이 실제로도 동일한 밝기로 인식되는 특성이 있어, 더 일관된 색상 팔레트를 만들 수 있습니다.

### 혁신 포인트
- OKLCH 색공간의 실용적 활용 (아직 널리 알려지지 않은 새로운 표준)
- APCA(Advanced Perceptual Contrast Algorithm) 지원으로 미래 웹 접근성 표준 선제 대응
- 3D 색공간 시각화를 통한 직관적인 색상 관계 이해

### 학습 기회
- OKLCH, LAB, sRGB 등 다양한 색공간의 특성과 변환 원리
- WCAG와 APCA 접근성 기준의 차이점과 계산 방법
- CSS Color Module Level 4의 새로운 색상 함수들 활용법

- **Coolors.co**: HSL 기반 팔레트 생성. OKLCH 지원 없음
- **Adobe Color**: 전문가용이지만 웹 개발자 친화적이지 않고 OKLCH 미지원

### 왜 이런 공백이 존재하는가?
OKLCH는 2020년에 표준화된 비교적 새로운 색공간이고, 브라우저 지원도 최근에야 완료되었습니다. 많은 개발자들이 아직 HSL에 익숙해서 OKLCH의 장점을 모르고 있습니다.

### 성공 지표
- [ ] 생성된 팔레트의 모든 색상이 지각적으로 균일한 밝기 분포를 가짐
- [ ] WCAG AA 기준을 만족하는 색상 조합 자동 생성
- [ ] CSS Color Level 4 지원 브라우저에서 정확한 색상 렌더링
