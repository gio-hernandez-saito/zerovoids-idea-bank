# D3 기반 컬러 팔레트 생성기

> OKLCH 색공간을 활용해 조화로운 컬러 팔레트를 시각적으로 생성하고 접근성을 검증하는 도구

## 🎯 문제 상황

### 현재 상황
디자이너와 개발자들은 컬러 팔레트를 만들 때 RGB나 HSL 색공간에 의존하지만, 이는 인간의 색 인지와 일치하지 않습니다. 기존 컬러 도구들은 수학적으로 조화로운 색상을 생성하지만, 실제 사용성과 접근성은 고려하지 않습니다. OKLCH처럼 인지적으로 균등한 색공간의 장점을 활용한 도구는 드뭅니다.

### 문제점들
- **색상 조화 부족**: HSL 기반 도구로 만든 팔레트가 시각적으로 불균형해 보임
- **접근성 검증 부재**: 색상 대비나 색각이상자 고려 없이 팔레트 생성
- **컬러 이론 이해 부족**: 보색, 삼색조화 등의 원리를 시각적으로 학습할 기회 부족

### 대상 사용자
디자인 시스템을 구축하는 프론트엔드 개발자, 브랜드 컬러를 정의하는 디자이너, 데이터 시각화용 색상 팔레트가 필요한 개발자

## 💡 해결책

### 핵심 개념
OKLCH 색공간에서 작동하는 인터랙티브 컬러 팔레트 생성기입니다. D3.js를 활용해 색상환과 명도/채도 공간을 3차원적으로 시각화하고, 사용자가 직관적으로 조화로운 색상을 선택할 수 있게 합니다. 생성된 팔레트는 실시간으로 접근성 기준(WCAG, APCA)을 검증하며, 다양한 색각이상 시뮬레이션을 제공합니다.

### 작동 방식
1. **색공간 탐색**: OKLCH 3D 원통을 D3로 시각화, 마우스로 색상 선택
2. **조화 규칙 적용**: 보색, 삼색조화, 단색조화 등의 컬러 이론 규칙을 시각적으로 표시
3. **팔레트 생성**: 선택된 기준색에서 자동으로 조화로운 색상들 생성
4. **접근성 검증**: 실시간으로 색상 대비비 계산 및 색각이상 시뮬레이션
5. **내보내기**: CSS 변수, Design Token, Figma 등 다양한 형태로 출력

### 주요 기능
- **3D 색공간 네비게이션**: OKLCH 원통을 직관적으로 탐색하며 색상 선택
- **실시간 조화 표시**: 선택된 색상에 대해 수학적으로 조화로운 색상들을 실시간 표시
- **접근성 대시보드**: WCAG AA/AAA, APCA 기준으로 모든 색상 조합의 대비비 매트릭스 표시
- **색각이상 시뮬레이션**: 프로타노피아, 듀테라노피아, 트리타노피아 시점에서 팔레트 미리보기
- **다양한 내보내기**: CSS Custom Properties, Sass 변수, Tailwind 설정, Design Token 등

## 🏗️ 기술 아키텍처

### 시스템 개요

```
┌─────────────────┐
│  Color Engine   │ (OKLCH 변환/계산)
└────────┬────────┘
         │
┌────────▼────────┐
│  D3 Renderer    │ (3D 색공간 시각화)
└────────┬────────┘
         │
┌────────▼────────┐

└────────┬────────┘
         │
┌────────▼────────┐
│ Export Module   │ (다양한 형식 출력)
└─────────────────┘
```

### 기술 스택 선택 이유
- **D3.js**: 복잡한 색공간 시각화와 인터랙티브한 3D 표현에 최적화됨. 색상 스케일과 보간 기능이 뛰어남
- **React**: 상태 관리와 컴포넌트 기반 UI 구성. D3와의 통합 패턴 학습 기회
- **TypeScript**: 색상 계산 로직의 타입 안전성 보장. OKLCH 값의 범위 검증
- **CSS3**: Custom Properties와 색공간 함수 활용으로 브라우저 네이티브 색상 처리

### 주요 기술적 도전
1. **OKLCH-RGB 변환 정확도**: 색공간 변환 시 정밀도 유지와 범위 클리핑 처리
2. **3D 인터랙션 UX**: 색상환을 3차원으로 표현하되 직관적인 조작 방법 설계
3. **실시간 성능**: 색상 변경 시 수백 개 대비비 계산의 최적화

## 🎨 독창성과 차별점

### 독특한 접근법
기존 컬러 도구들이 RGB/HSL에 머무르는 동안, 이 도구는 OKLCH라는 인지적 색공간을 중심으로 설계됩니다. 단순히 색상을 선택하는 것이 아니라 컬러 이론을 시각적으로 학습하면서 팔레트를 생성하는 교육적 도구이기도 합니다. D3.js의 강력한 시각화 기능으로 색공간 자체를 탐험 가능한 3D 환경으로 만든 점이 차별화 요소입니다.

### 혁신 포인트
- OKLCH 색공간의 3D 시각화를 통한 직관적 색상 선택
- 컬러 이론 규칙을 실시간 시각적 가이드로 표현
- 접근성 검증을 팔레트 생성 과정에 통합
- Design Token 시대에 맞는 다양한 출력 형식 지원

### 학습 기회
- OKLCH, LAB 등 고급 색공간 이해
- D3.js를 활용한 3D 데이터 시각화 기법
- 색상 접근성과 WCAG 기준에 대한 실무 지식
- React와 D3.js 통합 패턴

## 📊 시장성 및 검증

### 유사 프로젝트들
- **Adobe Color** (color.adobe.com): 웹 기반이지만 HSL/RGB 중심. OKLCH 지원 부족
- **Coolors.co**: 간편하지만 컬러 이론 설명과 접근성 검증 기능 제한적
- **Paletton**: 컬러 이론 기반이지만 구식 인터페이스, 3D 시각화 없음

### 기회가 존재하는 이유
OKLCH는 비교적 새로운 색공간으로 CSS Color Module Level 4에서 표준화되었지만, 이를 활용한 도구는 아직 부족합니다. 디자인 시스템과 접근성에 대한 관심이 높아지면서 체계적인 컬러 팔레트 도구의 수요가 증가하고 있습니다.

### 성공 지표
- [ ] OKLCH 색공간에서 조화로운 5색 팔레트 생성 가능
- [ ] 생성된 팔레트가 WCAG AA 기준 통과
- [ ] 색각이상 시뮬레이션에서도 구별 가능한 색상 조합
- [ ] CSS Custom Properties 형태로 내보내기 성공
