# Pitch Narrative Engine

> 야구 투구 시퀀스를 인터랙티브 스토리로 변환하여 전략의 숨겨진 드라마를 드러내는 도구

## 🎯 문제 상황

### 현재 상황
야구 분석은 대부분 통계 중심으로 이루어진다. 타율, ERA, WHIP 같은 숫자들이 성과를 측정하지만, 실제 경기에서 벌어지는 심리전과 전략적 드라마는 놓치고 있다. 특히 투수와 타자 간의 투구 시퀀스는 단순한 데이터 테이블로만 표현되어 그 안에 담긴 스토리가 보이지 않는다.

### 고충점
- **맥락 부족**: 개별 투구가 어떤 전략적 맥락에서 나왔는지 파악하기 어려움
- **패턴 인식의 한계**: 숫자로만 보면 투수의 심리전술이나 타자의 적응 패턴이 보이지 않음
- **스토리텔링 부재**: 코치나 팬에게 전략을 설명할 때 흥미롭게 전달하기 어려움

### 누가 이런 문제를 겪나?
야구 분석가, 코치진, 스포츠 미디어 기자, 그리고 야구의 전략적 측면에 관심 있는 팬들. 특히 데이터는 있지만 그것을 흥미롭게 전달해야 하는 사람들.

## 💡 해결책

### 핵심 개념
Pitch Narrative Engine은 투구 시퀀스 데이터를 받아서 자동으로 '스토리 아크'를 생성한다. 각 투구를 스토리의 한 장면으로 취급하고, 투수-타자 간의 심리전을 내러티브 구조로 재구성한다. "설정-갈등-절정-해결"의 고전적 스토리 구조를 야구 상황에 적용하여, 데이터를 드라마틱한 시각적 내러티브로 변환한다.

### 작동 방식
1. **데이터 입력**: Statcast나 CSV 형태의 투구 시퀀스 데이터 업로드
2. **내러티브 분석**: 투구 패턴, 카운트 변화, 구종 선택을 분석해 '스토리 포인트' 식별
3. **시각적 스토리 생성**: D3.js를 활용해 투구를 타임라인 상의 극적 순간들로 시각화
4. **오디오 내러티브**: 자동 생성된 텍스트를 음성으로 변환해 실제 중계 느낌 연출

### 주요 기능
- **스토리 아크 시각화**: 각 타석을 기승전결 구조로 표현하는 인터랙티브 차트
- **심리전 맵**: 투수가 타자를 어떻게 "설정"하고 "함정"에 빠뜨리는지 단계별 시각화
- **드라마틱 모멘트 하이라이트**: 결정적 순간(2스트라이크 후 변화구, 풀카운트 승부구 등)을 자동 감지
- **내러티브 템플릿**: 다양한 스토리텔링 스타일(스릴러, 코미디, 드라마) 적용 가능

## 🏗️ 기술 아키텍처

### 시스템 개요
```
┌─────────────┐
│ 데이터 파서  │ (CSV/JSON → 구조화된 시퀀스)
└──────┬──────┘
       │
┌──────▼──────┐
│스토리 엔진   │ (패턴 분석 + 내러티브 구조 매핑)
└──────┬──────┘
       │
┌──────▼──────┐
│ 시각화 엔진  │ (D3.js + 애니메이션)
└──────┬──────┘
       │
┌──────▼──────┐
│오디오 엔진   │ (Web Audio API + TTS)
└─────────────┘
```

### 기술 스택 선택 근거
- **D3.js**: 복잡한 시간축 기반 내러티브 시각화를 위한 최적 도구. 투구 타이밍과 강도를 시각적으로 표현
- **TypeScript**: 투구 데이터 구조가 복잡하므로 타입 안전성 필수. 내러티브 생성 로직의 복잡성 관리
- **Web Audio API**: 실시간 오디오 피드백으로 몰입도 증대. 투구 속도에 따른 사운드 이펙트

### 주요 기술 도전과제
1. **내러티브 알고리즘**: 어떤 투구 패턴이 "드라마틱"한가? 기계학습 없이 룰 기반으로 접근 vs 패턴 인식
2. **실시간 시각화 성능**: 9이닝 경기 전체를 부드럽게 애니메이션하면서도 인터랙티브 조작 가능하게

## 🎨 흥미로운 점

### 독특한 접근
데이터 시각화와 스토리텔링의 교차점을 탐구한다. 기존 야구 분석 도구들이 "무엇이 일어났는가"에 집중한다면, 이 도구는 "왜 그렇게 되었고, 어떤 드라마가 있었는가"에 초점을 맞춘다. 스포츠 데이터를 엔터테인먼트 콘텐츠로 변환하는 새로운 패러다임.

### 혁신 포인트
- 데이터 시각화에 내러티브 구조 이론 적용
- 스포츠 분석을 스토리텔링으로 접근하는 창의적 전환
- 시각적 요소와 오디오를 결합한 몰입형 데이터 경험
- 전문가용 분석 도구를 대중도 즐길 수 있는 엔터테인먼트로 확장

### 학습 기회
- D3.js의 복잡한 타임라인 애니메이션 구현
- 데이터에서 의미있는 패턴 추출하는 알고리즘 설계
- Web Audio API를 활용한 동적 사운드 디자인
- 스토리텔링 이론의 데이터 시각화 적용

## 📊 시장성 및 검증

### 유사 프로젝트
- **Baseball Savant** (mlb.com): 정적 통계 중심. 내러티브나 스토리텔링 요소 부재
- **FanGraphs** (fangraphs.com): 심화 분석이지만 전문가 대상. 일반인이 즐기기엔 진입장벽 높음
- **ESPN 야구 시각화**: 기본적인 차트. 투구 시퀀스의 드라마틱한 측면은 다루지 않음

### 이 틈새가 존재하는 이유
스포츠 데이터 시각화는 대부분 "정보 전달"에 집중해왔다. 하지만 스포츠는 본질적으로 드라마와 스토리다. 데이터 분석과 엔터테인먼트를 결합하려는 시도가 부족했고, 기술적으로도 웹에서 고품질 오디오-비주얼 경험을 구현하기 어려웠다.

### 성공 지표
- [ ] 5분 투구 시퀀스를 30초 내러티브로 압축 가능
- [ ] 비야구팬도 "재미있다"고 반응하는 콘텐츠 생성
- [ ] 야구 커뮤니티에서 공유되고 토론거리가 되는 결과물
