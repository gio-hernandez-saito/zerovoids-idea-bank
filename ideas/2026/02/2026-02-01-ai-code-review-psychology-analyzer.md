# AI 코드 리뷰 심리 분석기

> 코드 리뷰 댓글의 감정적 톤을 분석하여 팀의 심리적 건강도를 시각화하는 도구

## 🎯 문제 상황

### 현재 상황
코드 리뷰는 개발 프로세스의 핵심이지만, 감정적 측면은 완전히 무시되고 있습니다. "이 코드는 형편없다", "다시 작성해라" 같은 직접적인 피드백이 개발자들의 자존감과 팀 분위기에 미치는 영향을 측정할 방법이 없습니다. 많은 개발자들이 코드 리뷰로 인한 스트레스를 경험하지만, 이는 보이지 않는 문제로 남아있습니다.

### 고충 포인트
- **감정적 피로**: 날카로운 리뷰 댓글로 인한 개발자 번아웃
- **팀 분위기 악화**: 공격적인 피드백 문화로 인한 소통 위축
- **생산성 저하**: 스트레스로 인한 코드 품질 및 속도 감소
- **무의식적 패턴**: 리뷰어들이 자신의 톤을 객관적으로 파악하기 어려움

### 누가 이 문제를 겪고 있는가?
코드 리뷰 문화가 활발한 개발팀, 특히 주니어 개발자가 많은 팀이나 원격 근무로 인해 비언어적 소통이 제한된 환경의 개발자들

## 💡 해결책

### 핵심 컨셉
AI 코드 리뷰 심리 분석기는 GitHub, GitLab 등의 코드 리뷰 댓글을 실시간으로 분석하여 감정적 톤, 건설성, 스트레스 지수를 측정합니다. 팀의 심리적 건강 상태를 대시보드로 시각화하고, 개선이 필요한 패턴을 조기에 감지하여 더 건강한 코드 리뷰 문화를 만들어갑니다. 단순한 감정 분석을 넘어 개발 맥락을 이해하는 전문화된 AI를 활용합니다.

### 작동 방식
1. **데이터 수집**: GitHub API를 통해 PR 댓글 및 리뷰 데이터 수집
2. **AI 분석**: OpenAI API로 댓글의 감정 톤, 건설성, 공격성 수준 분석
3. **패턴 감지**: 시간별, 개발자별, 프로젝트별 트렌드 및 이상 징후 탐지
4. **시각화**: D3.js로 팀 심리 건강 지표를 직관적인 차트로 표현
5. **알림**: 스트레스 급증 또는 부정적 패턴 감지 시 팀 리더에게 알림

### 주요 기능
- **실시간 감정 분석**: 각 리뷰 댓글의 감정적 톤을 즉시 분석하고 점수화
- **팀 건강 대시보드**: 전체 팀의 심리적 건강 상태를 한눈에 파악할 수 있는 시각적 대시보드
- **개인별 패턴 인사이트**: 개발자별 리뷰 스타일 분석 및 개선 포인트 제안
- **건설적 리뷰 가이드**: AI가 제안하는 더 나은 피드백 표현 방법
- **스트레스 얼리 워닝**: 번아웃 위험이 높은 개발자 조기 감지 시스템

- **D3.js**: 심리적 데이터의 복잡한 패턴을 직관적으로 시각화할 수 있는 유연성 제공

- **Vite**: 빠른 개발 환경과 번들링으로 실시간 데이터 업데이트 최적화

### 주요 기술적 과제
1. **맥락적 감정 분석**: "이 코드는 개선이 필요합니다"와 "이 코드는 쓰레기네요"의 차이를 정확히 구분
2. **실시간 데이터 처리**: GitHub 웹훅과 연동하여 댓글 작성 즉시 분석 결과 반영
3. **프라이버시 보호**: 민감한 심리적 데이터의 안전한 처리 및 익명화

### 독특한 접근법
기존의 코드 품질 분석 도구들은 기술적 측면에만 집중했지만, 이 프로젝트는 개발 프로세스의 **인간적 측면**을 조명합니다. 코드 리뷰를 단순한 기술적 검토가 아닌 팀 커뮤니케이션의 핵심 채널로 바라보는 새로운 관점입니다. 개발자의 웰빙과 팀 문화를 정량적으로 측정할 수 있다는 점이 혁신적입니다.

### 혁신 포인트
- 개발 도구와 심리학의 융합: 기술과 인문학적 관점의 창의적 결합
- 예방적 번아웃 관리: 문제가 심각해지기 전에 조기 감지
- AI의 맥락적 이해: 단순 키워드 매칭이 아닌 개발 상황을 고려한 분석
- 데이터 기반 팀 문화 개선: 주观적 느낌을 객관적 지표로 전환

### 학습 기회
- OpenAI API의 고급 프롬프트 엔지니어링 기법
- 복잡한 시계열 데이터의 패턴 인식 알고리즘
- 민감한 데이터의 윤리적 처리 및 프라이버시 보호
- 심리학적 지표의 시각화 디자인 원칙
