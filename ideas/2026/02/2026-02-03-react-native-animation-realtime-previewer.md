# React Native 애니메이션 실시간 프리뷰어

> React Native 애니메이션을 코드 변경과 동시에 브라우저에서 실시간으로 확인하세요

## 🎯 문제 상황

### 현재 상황
React Native에서 애니메이션을 개발할 때 개발자들은 끔찍한 피드백 루프에 갇혀 있습니다. 코드를 수정하고, Metro 번들러가 다시 빌드하고, 시뮬레이터나 실기기에서 확인하는 과정을 수십 번 반복해야 합니다. 특히 복잡한 애니메이션일수록 미세한 타이밍이나 이징 조정을 위해 이 과정을 무한히 반복하게 됩니다.

### 핵심 고충
- **긴 피드백 사이클**: 애니메이션 하나 조정하는데 30초~1분씩 대기
- **컨텍스트 손실**: 빌드 중에 다른 작업을 하다 보면 원래 의도를 잊어버림
- **비교 어려움**: 이전 버전과 현재 버전을 동시에 비교할 수 없음
- **매개변수 실험의 어려움**: duration, delay, easing 등을 실시간으로 조정 불가

### 누가 겪고 있나?
React Native로 인터랙티브한 앱을 만드는 모든 개발자, 특히 복잡한 UI 애니메이션이나 제스처 기반 인터랙션을 구현하는 팀들이 매일 겪는 문제입니다.

## 💡 해결 방안

### 핵심 컨셉
React Native 애니메이션 실시간 프리뷰어는 React Native 애니메이션 코드를 웹 환경에서 실시간으로 렌더링하는 도구입니다. 코드를 수정하는 순간 브라우저에서 애니메이션이 즉시 업데이트되며, 슬라이더와 컨트롤을 통해 애니메이션 매개변수를 실시간으로 조정할 수 있습니다. React Native의 Animated API를 웹용 CSS 애니메이션으로 변환해 네이티브와 거의 동일한 결과를 보여줍니다.

### 작동 방식
1. **애니메이션 코드 감지**: Metro 번들러와 연동해 Animated API 사용 컴포넌트를 식별
2. **실시간 변환**: React Native 애니메이션을 CSS 애니메이션/Web Animations API로 실시간 변환
3. **브라우저 프리뷰**: 웹 인터페이스에서 애니메이션을 즉시 렌더링하고 인터랙티브 컨트롤 제공
4. **양방향 동기화**: 웹에서 조정한 값을 다시 코드에 반영하는 옵션 제공

### 주요 기능
- **즉시 프리뷰**: 코드 저장과 동시에 브라우저에서 애니메이션 업데이트
- **인터랙티브 컨트롤**: duration, delay, easing을 슬라이더로 실시간 조정
- **비교 모드**: 여러 버전의 애니메이션을 나란히 비교
- **성능 시각화**: 애니메이션 성능 메트릭을 실시간 그래프로 표시

## 🏗️ 기술 아키텍처

### 시스템 개요
```
┌─────────────────┐    WebSocket     ┌─────────────────┐
│  Metro Plugin   │◄─────────────────►│   Web Server    │
│ (코드 감시/변환)  │                 │  (변환 엔진)      │
└─────────────────┘                 └─────────┬───────┘
                                              │
                                     ┌────────▼────────┐
                                     │ React 프리뷰어   │
                                     │ (브라우저 UI)    │
                                     └─────────────────┘
```

### 기술 스택 근거
- **Metro Plugin**: React Native 빌드 프로세스에 직접 통합되어 코드 변경을 실시간 감지
- **WebSocket**: 코드 변경사항을 브라우저로 즉시 전송하는 실시간 통신
- **TypeScript**: React Native Animated API 타입 정의를 활용한 정확한 코드 파싱
- **React**: 프리뷰 UI를 빠르게 구축하고 상태 관리

### 주요 기술적 도전
1. **API 변환의 정확성**: React Native Animated API를 CSS/Web Animations로 1:1 매핑하기. Timing 함수와 interpolation 로직을 정확히 재현해야 함.
2. **성능 최적화**: 코드 변경마다 실시간 파싱과 변환이 일어나므로 debouncing과 효율적인 diff 알고리즘 필요.

## 🎨 무엇이 흥미로운가?

React Native 개발 도구들은 대부분 디바이스에서의 디버깅에 초점을 맞추지만, 이 도구는 웹 환경을 활용해 더 빠른 피드백 루프를 만듭니다. React Native 애니메이션을 웹으로 '번역'하는 아이디어는 기존에 시도되지 않았던 접근법입니다.

### 혁신 포인트

- 실시간 코드-프리뷰 동기화 시스템
- 애니메이션 매개변수의 인터랙티브 조정
- 크로스 플랫폼 애니메이션 호환성 검증

### 학습 기회
- React Native Animated API의 내부 동작 원리
- Web Animations API와 CSS 애니메이션의 고급 활용
- Metro 번들러 플러그인 개발
- 실시간 개발 도구 아키텍처
