# React Native 애니메이션 AI 생성기

> 자연어 설명만으로 React Native 애니메이션 코드를 즉시 생성하고 실시간 미리보기까지

## 🎯 문제 상황

### 현재 상황
React Native에서 복잡한 애니메이션을 구현하려면 Reanimated의 복잡한 API를 완전히 이해해야 합니다. useSharedValue, withTiming, interpolate 등의 개념들이 서로 얽혀 있어 원하는 애니메이션을 만들기까지 시간이 오래 걸립니다. 특히 여러 요소가 연결된 복합 애니메이션은 더욱 어렵습니다.

### 문제점들
- **높은 학습 곡선**: Reanimated 3의 복잡한 API와 개념들을 모두 익혀야 함
- **시행착오 반복**: 원하는 효과를 얻기까지 무수한 테스트와 수정 필요
- **참고 자료 부족**: 구체적인 애니메이션 구현 예제가 제한적
- **시간 소모**: 간단한 애니메이션도 구현하는데 몇 시간씩 소요

### 누가 겪는 문제인가?
React Native 개발자들, 특히 애니메이션 경험이 적은 개발자나 디자이너가 구상한 복잡한 모션을 구현해야 하는 프론트엔드 개발자들이 이 문제를 겪습니다.

## 💡 해결 방안

### 핵심 개념
React Native 애니메이션 AI 생성기는 자연어로 설명한 애니메이션을 실제 Reanimated 3 코드로 변환해주는 도구입니다. "버튼을 누르면 카드가 위로 슬라이드되면서 페이드 인" 같은 설명을 입력하면, 즉시 실행 가능한 TypeScript 코드를 생성하고 실시간 미리보기를 제공합니다. AI가 애니메이션 패턴을 학습하여 최적화된 코드를 생성합니다.

### 작동 방식
1. **자연어 입력**: 원하는 애니메이션을 일반 언어로 설명
2. **AI 분석**: Claude API가 설명을 분석하여 애니메이션 구조 파악
3. **코드 생성**: 최적화된 Reanimated 3 TypeScript 코드 자동 생성
4. **실시간 미리보기**: 생성된 코드를 즉시 Expo 환경에서 실행하여 결과 확인
5. **코드 내보내기**: 완성된 코드를 클립보드로 복사하거나 파일로 저장

### 주요 기능
- **스마트 코드 생성**: 타이밍, 이징, 인터폴레이션을 자동으로 최적화하여 자연스러운 애니메이션 코드 생성
- **실시간 미리보기**: 생성된 코드를 바로 실행하여 시각적 결과 확인 가능
- **코드 설명**: 생성된 코드의 각 부분이 어떤 역할을 하는지 한국어로 주석 제공
- **애니메이션 라이브러리**: 자주 사용되는 패턴들을 템플릿으로 저장하고 재사용
