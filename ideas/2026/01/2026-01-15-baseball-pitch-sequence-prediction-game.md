# 야구 투구 시퀀스 예측 게임

> 실제 야구 데이터로 투수의 다음 구종을 맞춰보는 인터랙티브 게임

## 🎯 문제 상황

### 현재 상황
야구 팬들은 경기를 보면서 "다음엔 뭘 던질까?"라고 예측하며 재미를 느낍니다. 하지만 이런 예측 능력을 체계적으로 연습하고 검증할 수 있는 도구는 거의 없습니다. 기존 야구 분석 도구들은 정적인 통계 정보만 제공할 뿐입니다.

### 핵심 문제점
- **수동적 관람**: 야구 시청이 단순히 결과를 보는 일방향적 경험
- **직관 검증 불가**: 자신의 야구 센스가 얼마나 정확한지 알 수 없음
- **학습 기회 부족**: 투수의 패턴과 전략을 체계적으로 이해할 방법 없음

### 대상 사용자
야구를 좋아하고 경기의 전략적 측면에 관심이 있는 팬들. 특히 투수의 구종 선택 패턴을 이해하고 싶어하는 중급 이상의 야구 팬들.

---

## 💡 해결 방안

### 핵심 개념
실제 메이저리그 투구 데이터를 기반으로 한 예측 게임입니다. 사용자는 실제 상황(카운트, 주자, 이닝 등)을 보고 투수가 다음에 던질 구종을 예측합니다. 예측 후 실제 투구를 확인하고 점수를 얻는 방식으로, 야구 관람을 능동적이고 교육적인 경험으로 바꿔줍니다.

### 작동 방식
1. **상황 제시**: 실제 경기의 특정 순간(카운트, 주자 상황, 이닝, 타자 정보)
2. **예측 입력**: 사용자가 구종(직구, 슬라이더, 커브 등)과 위치를 선택
3. **결과 공개**: 실제 투구를 애니메이션으로 보여주고 정답 여부 확인
4. **학습 제공**: 해당 상황에서의 투수 패턴과 선택 이유 설명

### 주요 기능
- **실시간 스코어링**: 연속 정답 수, 정확도, 난이도별 점수
- **패턴 분석**: 사용자의 예측 성향과 약점 분석
- **투수별 모드**: 특정 투수의 패턴을 집중 학습
- **상황별 연습**: 2스트라이크, 득점권 등 특정 상황 집중 훈련

```

       │
```

### 기술 스택 선택 이유
- **React + TypeScript**: 복잡한 게임 상태 관리와 컴포넌트 재사용성
- **D3.js**: 투구 궤적 애니메이션과 통계 시각화
- **CSS3 애니메이션**: 부드러운 UI 전환과 투구 모션
- **Vite**: 빠른 개발 환경과 번들링

### 핵심 기술 과제
1. **투구 애니메이션**: 실제와 유사한 궤적과 타이밍 구현
2. **데이터 최적화**: 대량의 투구 데이터를 효율적으로 필터링하고 로딩
3. **예측 알고리즘**: 사용자 패턴 학습과 개인화된 난이도 조절
