# 컴포넌트 족보 탐험기

> React 컴포넌트들의 복잡한 가족관계를 인터랙티브한 족보로 탐험하세요

## 🎯 문제 상황

### 현재 상황
대형 React 프로젝트에서 컴포넌트 간의 관계를 파악하는 것은 마치 복잡한 가계도를 머릿속으로 그리는 것과 같습니다. 개발자들은 "이 컴포넌트가 어디서 사용되는지", "어떤 props가 어디서 내려오는지", "이 컴포넌트를 수정하면 어떤 자식들에게 영향을 미칠지"를 파악하기 위해 수십 개의 파일을 왔다 갔다 하며 시간을 낭비합니다.

### 고충 포인트
- **정신적 스택 오버플로우**: 컴포넌트 A가 B를 쓰고, B가 C를 쓰고... 5단계만 내려가도 머리가 터집니다
- **props 추적의 악몽**: `user.profile.avatar.url`이 도대체 몇 단계를 거쳐 내려오는지 알 수 없음
- **리팩토링 공포증**: 이 컴포넌트를 건드리면 어디가 망가질지 예상 불가능
- **온보딩 지옥**: 신입 개발자가 컴포넌트 구조를 이해하는 데 몇 주가 걸림

### 누가 이 문제를 겪나요?
100개 이상의 컴포넌트를 가진 React 프로젝트에서 일하는 프론트엔드 개발자들. 특히 디자인 시스템이나 공통 컴포넌트 라이브러리를 관리하는 팀, 그리고 레거시 코드베이스를 유지보수하는 개발자들이 가장 심각하게 경험합니다.

## 💡 해결책

### 핵심 컨셉
컴포넌트 족보 탐험기는 React 컴포넌트 트리를 실제 가족 족보처럼 시각화하는 인터랙티브 도구입니다. 각 컴포넌트를 가족 구성원으로, props를 유전자로, context를 가문의 전통으로 표현하여 복잡한 관계를 직관적으로 이해할 수 있게 합니다. 드래그앤드롭으로 족보를 재배열하고, 특정 "혈통"을 따라 props가 어떻게 전달되는지 애니메이션으로 추적할 수 있습니다.

### 작동 방식
1. **프로젝트 스캔**: AST 파서가 React 코드를 분석하여 컴포넌트 관계 추출
2. **족보 생성**: D3.js로 계층적 트리 레이아웃을 생성, 각 컴포넌트를 노드로 배치
3. **인터랙티브 탐험**: 클릭하면 상세 정보 표시, 드래그로 시점 이동, props 흐름을 애니메이션으로 시각화
4. **관계 분석**: 특정 컴포넌트의 "조상", "후손", "형제" 관계를 하이라이트

### 핵심 기능
- **실시간 족보 시각화**: 컴포넌트 구조를 가계도 스타일로 표현, 부모-자식 관계를 직관적으로 파악
- **props 혈통 추적**: 특정 prop이 어떤 경로로 전달되는지 애니메이션으로 표시
- **컴포넌트 프로필**: 각 컴포넌트의 "신상정보" - 사용 빈도, props 타입, 자식 수 등
- **영향도 분석**: 특정 컴포넌트를 수정했을 때 영향받는 "친척들"을 시각적으로 표시
- **시간여행 모드**: 컴포넌트 구조의 변화를 Git 커밋별로 비교

## 🏗️ 기술 아키텍처

### 시스템 개요

```
┌─────────────────┐
│   AST Parser    │ (React 코드 분석)
└────────┬────────┘
         │
┌────────▼────────┐
│ Component Graph │ (관계 모델링)
└────────┬────────┘
         │
┌────────▼────────┐
│ D3.js Renderer  │ (족보 시각화)
└────────┬────────┘
         │
┌────────▼────────┐
│ React UI Shell  │ (인터랙션 레이어)
└─────────────────┘
```

### 기술 스택 근거
- **TypeScript AST Parser**: React 컴포넌트와 JSX를 정확히 파싱하기 위해 기존 도구 활용 (ts-morph)
- **D3.js**: 계층적 데이터 시각화에 최적화된 라이브러리, 트리/포스 레이아웃 알고리즘 내장
- **React**: 도구 자체도 React로 구현하여 메타적 일관성 확보, 상태 관리 용이
- **Vite**: 빠른 개발 환경과 HMR로 실시간 피드백 지원

### 주요 기술 과제
1. **대용량 컴포넌트 트리 렌더링**: 100+ 컴포넌트를 부드럽게 렌더링하기 위한 가상화 및 LOD(Level of Detail) 시스템 구현
2. **실시간 props 추적**: props가 여러 단계를 거치며 변형되는 과정을 정확히 추적하는 알고리즘 설계

### 혁신 포인트

- 시간여행 기능으로 컴포넌트 진화 과정 추적
- 드래그앤드롭 인터랙션으로 능동적 탐험 경험

### 학습 기회
- D3.js 계층적 레이아웃과 애니메이션 마스터
- AST 파싱과 코드 분석 기법 습득
- 대용량 데이터 시각화 최적화 전략
- 메타포를 활용한 UX 디자인 패턴
