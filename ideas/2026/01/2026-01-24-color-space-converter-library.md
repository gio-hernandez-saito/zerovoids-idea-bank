# 컬러 스페이스 변환 라이브러리

> HSL, HCT, OKLCH, RGB 간 정확한 색상 변환과 접근성 검증을 한 번에

## 🎯 문제 상황

### 현재 상황
디자인 시스템과 접근성을 고려한 웹 개발에서 색상 작업은 복잡합니다. RGB에서 HSL로, OKLCH에서 HEX로, 또는 APCA 대비를 위한 HCT 변환 등 다양한 색상 공간 간 변환이 필요하지만, 기존 라이브러리들은 제한적이거나 부정확합니다.

### 고충 사항
- **변환 부정확성**: 기존 라이브러리들의 색상 변환 결과가 미묘하게 다름
- **접근성 계산 혼란**: WCAG 2.1과 APCA 기준이 다른데 어떤 것을 써야 할지 모호함
- **분산된 도구**: 색상 변환, 대비 계산, 팔레트 생성을 각각 다른 패키지로 해야 함

### 누가 겪는 문제인가?
디자인 시스템을 구축하는 프론트엔드 개발자, 접근성을 고려한 웹 개발자, 색상 이론을 코드로 구현하려는 개발자들이 겪는 문제입니다.

## 💡 해결 방안

### 핵심 개념
모든 주요 색상 공간(RGB, HSL, OKLCH, HCT) 간 정확한 변환을 제공하고, WCAG와 APCA 접근성 기준을 모두 지원하는 TypeScript 라이브러리입니다. 단순한 변환뿐만 아니라 색상 팔레트 생성, 접근성 검증, 색상 하모니 계산까지 포함합니다.

### 작동 방식
1. **색상 입력**: 어떤 형식으로든 색상을 입력 (hex, rgb, hsl, oklch 등)
2. **내부 정규화**: 모든 색상을 내부적으로 정확한 수학적 모델로 변환
3. **원하는 출력**: 필요한 색상 공간이나 접근성 정보로 변환하여 반환

### 주요 기능
- **정확한 변환**: CIE LAB 기반 정확한 색상 공간 변환
- **접근성 통합**: WCAG 2.1과 APCA 대비 계산을 모두 지원
- **팔레트 생성**: OKLCH 기반 일관된 명도/채도의 색상 팔레트 자동 생성
- **색맹 시뮬레이션**: Protanopia, Deuteranopia, Tritanopia 시뮬레이션

### 시스템 개요

```
┌─────────────────┐
│   파서 레이어    │ (색상 형식 감지 & 파싱)
└────────┬────────┘
         │
┌────────▼────────┐
│  변환 엔진       │ (CIE LAB 기반 정확한 변환)
└────────┬────────┘
         │
┌────────▼────────┐
│  접근성 모듈     │ (WCAG, APCA 계산)
└────────┬────────┘
         │
┌────────▼────────┐
│   유틸리티      │ (팔레트, 하모니, 시뮬레이션)
└─────────────────┘
```

### 기술 스택 근거
- **TypeScript**: 색상 타입 안전성과 API 자동완성 제공
- **Vite**: 빠른 개발 환경과 최적화된 번들링
- **Vitest**: 색상 변환의 정확성을 보장하는 광범위한 테스트 필요

### 주요 기술적 도전
1. **색상 변환 정확성**: OKLCH와 HCT 변환에서 수학적 정밀도 확보. Delta E 기반 검증 테스트 구현
2. **성능 최적화**: 대량의 색상 변환 시에도 빠른 처리 속도. 룩업 테이블과 캐싱 전략 필요

## 📊 시장 & 검증

### 유사 프로젝트
- **chroma.js** (github.com/gka/chroma.js): 색상 조작 라이브러리지만 OKLCH 미지원, 접근성 기능 부족
- **culori** (github.com/Evercoder/culori): 다양한 색상 공간 지원하지만 접근성 통합 없음, API가 복잡
- **color** (npmjs.com/package/color): 기본적인 변환만 제공, 최신 색상 공간 미지원

### 이런 공백이 존재하는 이유
OKLCH와 HCT는 비교적 최근에 주목받기 시작했고, 접근성 요구사항도 계속 발전하고 있습니다. 기존 라이브러리들은 하위 호환성 때문에 새로운 표준을 적극 도입하기 어려운 상황입니다.

### 성공 지표
- [ ] Delta E < 1.0의 색상 변환 정확도 달성
- [ ] npm에서 월 1000+ 다운로드 달성
- [ ] 디자인 시스템 도구에서 실제 사용 사례 3개 확보
